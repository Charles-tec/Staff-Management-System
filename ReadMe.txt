Usage Notes 
-----------------------------------------------------------------------------
-This application captures, edits, deletes, updates employees of an organisation for different 
departments 
-To use run the Staff Management System.jar file 
---------------------------------------------------------------------------------------



installation notes 
---------------------------------------------------------------------------
-To run the project 
	-project is available on github url ->https://github.com/Charles-tec/Staff-Management-System.git
	-Download and install java 8 
	-Use either Eclipse or Intellij IDE (any version)
	-Create database using workbench or Xammp 
	-Database being used is called staffdatabase
